# Zone Monitor Project (YOLOv8 + OpenCV)

Система мониторинга запрещенных зон на основе видео. Использует Ultralytics YOLOv8 для детекции и трекинга людей (класс `person`), а также проверку проникновения в полигональные зоны, заданные пользователем.

## Возможности
- **Несколько запрещённых зон** — каждая со своим именем и цветом
- **Встроенный трекер** — стабильные ID людей между кадрами (BoT-SORT)
- **Тревога "ALARM!"** с задержкой 3 секунды после выхода из зоны
- **Автоматическое логирование** — CSV-файл с метками времени, ID, именем зоны
- **Скриншоты инцидентов** — автоматически сохраняются в папку `incidents/`

## Структура проекта
```
zone_monitor/
├── config.py              # Настройки (пути, цвета, пороги)
├── zone_selector.py       # Разметка зон мышкой (поддержка нескольких)
├── main.py                # Основной цикл: YOLO + трекинг + тревога
├── logger.py              # Логирование инцидентов (CSV + скриншоты)
├── requirements.txt       # Зависимости
├── restricted_zones.json  # Координаты зон (создаётся автоматически)
└── incidents/             # Папка со скриншотами и CSV-логом
    ├── incidents.csv
    └── *.jpg
```

## Запуск

### Шаг 1: Разметка зон
```bash
python zone_selector.py
```
| Клавиша       | Действие                              |
|---------------|---------------------------------------|
| ЛКМ           | Добавить точку полигона               |
| `Enter`/`Space` | Завершить текущую зону, начать новую |
| `C`           | Очистить текущий (незавершённый) полигон |
| `S`           | Сохранить все зоны и выйти            |

### Шаг 2: Мониторинг
```bash
python main.py
```
- Человек в зоне → рамка цветом зоны + `ALARM!`
- Человек вышел → тревога держится ещё 3 сек
- Инциденты пишутся в `incidents/incidents.csv` + скриншоты
- Выход: `q`
