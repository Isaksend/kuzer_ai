# Zone Monitor Project (YOLOv8 + OpenCV)

Система мониторинга запрещенных зон на основе видео. Использование Ultralytics YOLOv8 для детекции и трекинга людей (класс `person`), а также проверка проникновения в полигональную зону, заданную пользователем.

## Структура проекта
* `config.py` - Файл конфигурации (пути, цвета, задержка).
* `zone_selector.py` - Модуль разметки. Позволяет задать многоугольник (запрещенную зону) мышкой на первом кадре видео.
* `main.py` - Основной скрипт. Выполняет инференс YOLO, геометрическую проверку (вхождение в полигон) и логику трекинга задержки тревоги (3 секунды).
* `restricted_zones.json` - Сохраняет координаты размеченной зоны.

## Инструкция по установке

1. Создайте виртуальное окружение (рекомендуется) и активируйте его:
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/MacOS:
source venv/bin/activate
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Подготовьте видео для анализа и назовите его `test.mp4` (или измените `VIDEO_SOURCE` в `config.py`). Положите видео в корень проекта.

## Запуск

**Шаг 1: Разметка зоны**
Запустите интерфейс разметки:
```bash
python zone_selector.py
```
- Кликните левой кнопкой мыши по кадру, чтобы поставить точки полигона.
- Нажмите `c` чтобы очистить точки.
- Нажмите `Enter` или `Пробел`, чтобы сохранить зону и выйти.

**Шаг 2: Мониторинг и Трекинг**
Запустите основной скрипт:
```bash
python main.py
```
- Дождитесь загрузки весов модели `yolov8n.pt`.
- Если человек (ID трекера) попадает в размеченную зону, на экране появится надпись `ALARM!`.
- После выхода человека из кадра или из зоны, надпись `ALARM!` будет держаться на экране ровно 3 секунды.
- Для выхода нажмите `q`.
